# How to build
1. Open solution_private.targets, paste path to game and save
1. Open visual studio (tested with 2022)
1. Install harmony via NuGet package manager (google how to do it xD)
1. Right-click on mod project -> choose build
1. In Output log you will see where the zip with mode is
1. GLHF :)
